# Product Overview

AI Integration Platform для автоматизации бизнес-процессов в сфере продовольствия и FMCG.

## Core Purpose

Автоматизация работы с лидами, генерация коммерческих предложений и синхронизация данных между Bitrix24 CRM и 1С с использованием AI (GPT-4, Claude).

## Key Features

- AI-анализ лидов с оценкой качества (1-10) и рекомендациями
- Автоматическая генерация персонализированных коммерческих предложений
- Двусторонняя синхронизация данных между Bitrix24 и 1С
- Массовые операции (batch обработка лидов и сделок)
- Webhook интеграция для автоматической обработки событий
- Три способа интеграции с Bitrix24: REST API, JS SDK, MCP (Model Context Protocol)

## Target Users

- Менеджеры по продажам (работа с лидами и КП)
- Администраторы CRM (настройка автоматизации)
- Разработчики (расширение функционала)

## Business Value

- Экономия 45+ часов в неделю на рутинных операциях
- Увеличение конверсии лидов на 25%
- Ускорение создания КП в 6 раз (с 30-60 мин до 10 сек)
- Устранение ошибок при синхронизации данных
