# Библиотека промптов для сотрудников

## Промпты для работы с лидами

### Анализ качества лида
```
Проанализируй лид со следующими данными:
- Компания: [название]
- Контакт: [имя]
- Запрос: [описание]
- Источник: [откуда пришел]

Оцени:
1. Качество лида (1-10)
2. Вероятность сделки (%)
3. Рекомендуемые действия
4. Риски
```

### Определение потребностей клиента
```
На основе переписки с клиентом определи:
- Основные потребности
- Бюджет (если упоминается)
- Сроки принятия решения
- Ключевые критерии выбора
- Возможные возражения

Переписка:
[вставить текст переписки]
```

## Промпты для создания контента

### Коммерческое предложение
```
Создай коммерческое предложение для:
- Клиент: [название компании]
- Сфера деятельности: [описание]
- Товары: [список товаров]
- Особые условия: [если есть]

Структура КП:
1. Приветствие
2. Описание товаров
3. Цены и условия
4. Преимущества работы с нами
5. Призыв к действию
```

### Ответ на запрос клиента
```
Клиент написал: "[текст запроса]"

Создай профессиональный ответ, который:
- Отвечает на все вопросы
- Подчеркивает наши преимущества
- Содержит призыв к действию
- Написан дружелюбным тоном
```

### Email-рассылка
```
Создай текст email-рассылки для:
- Целевая аудитория: [описание]
- Цель: [что хотим получить]
- Акция/предложение: [описание]
- Тон: [деловой/дружеский/информационный]

Требования:
- Цепляющая тема письма
- Короткий и понятный текст
- Четкий призыв к действию
```

## Промпты для анализа данных

### Анализ отзыва клиента
```
Проанализируй отзыв клиента:
"[текст отзыва]"

Определи:
- Тональность (позитивная/негативная/нейтральная)
- Ключевые проблемы или похвалы
- Рекомендации по улучшению
- Нужно ли срочное реагирование
```

### Сегментация клиентов
```
На основе данных о клиенте:
- Объем закупок: [сумма]
- Частота заказов: [количество]
- Категории товаров: [список]
- История работы: [период]

Определи:
- Сегмент клиента (A/B/C)
- Потенциал роста
- Рекомендуемая стратегия работы
```

## Промпты для обучения

### Создание инструкции
```
Создай пошаговую инструкцию для сотрудников:
Тема: [название процесса]
Целевая аудитория: [кто будет использовать]

Требования:
- Простой язык
- Четкие шаги
- Примеры
- Что делать при ошибках
```

### Подготовка к встрече
```
Помоги подготовиться к встрече с клиентом:
- Компания: [название]
- Сфера: [деятельность]
- Цель встречи: [описание]
- Известные потребности: [список]

Подготовь:
1. Ключевые вопросы для клиента
2. Наши преимущества для этого клиента
3. Возможные возражения и ответы
4. Варианты предложений
```

## Промпты для автоматизации

### Классификация входящих запросов
```
Классифицируй запрос:
"[текст запроса]"

Категории:
- Новый заказ
- Вопрос по товару
- Жалоба
- Запрос документов
- Изменение заказа
- Другое

Также определи срочность: высокая/средняя/низкая
```

### Извлечение данных из текста
```
Извлеки структурированные данные из сообщения:
"[текст сообщения]"

Найди и выдели:
- Название компании
- Контактное лицо
- Телефон
- Email
- Интересующие товары
- Объемы
- Сроки
```

## Советы по использованию промптов

1. **Будьте конкретны**: Чем точнее запрос, тем лучше результат
2. **Добавляйте контекст**: Укажите специфику вашего бизнеса
3. **Итерируйте**: Если результат не устраивает, уточните запрос
4. **Проверяйте факты**: AI может ошибаться, всегда проверяйте важную информацию
5. **Сохраняйте удачные промпты**: Создавайте свою библиотеку работающих запросов

## Примеры комбинирования промптов

### Полный цикл работы с лидом

1. Анализ лида (промпт выше)
2. Подготовка к звонку (промпт выше)
3. Создание КП (промпт выше)
4. Написание follow-up письма (промпт выше)

Используйте промпты последовательно для комплексной работы с клиентом.
