# Интеграция с Bitrix24 через MCP (Model Context Protocol)

## Что такое MCP для Bitrix24?

Model Context Protocol (MCP) - это стандартизированный протокол для интеграции AI-инструментов с внешними системами. Bitrix24 MCP сервер предоставляет прямой доступ к Bitrix24 API через AI-ассистентов типа Claude, ChatGPT, Cursor и других.

## Преимущества MCP

- **Прямая интеграция**: AI может напрямую работать с Bitrix24 без промежуточного кода
- **Контекстная работа**: AI понимает структуру данных Bitrix24
- **Автоматизация**: Создание сложных сценариев через естественный язык
- **Безопасность**: Контроль доступа через настройки MCP

## Установка Bitrix24 MCP сервера

### Вариант 1: Через uvx (рекомендуется)

```bash
# Установка uv (если еще не установлен)
# Windows (PowerShell)
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"

# Linux/macOS
curl -LsSf https://astral.sh/uv/install.sh | sh

# Запуск Bitrix24 MCP сервера
uvx bitrix24-mcp-server
```

### Вариант 2: Через npm

```bash
npm install -g @bitrix24/mcp-server
bitrix24-mcp-server
```

## Настройка в Kiro

Создайте или отредактируйте файл `.kiro/settings/mcp.json`:

```json
{
  "mcpServers": {
    "bitrix24": {
      "command": "uvx",
      "args": ["bitrix24-mcp-server"],
      "env": {
        "BITRIX24_WEBHOOK_URL": "https://your-domain.bitrix24.ru/rest/1/xxxxx/",
        "BITRIX24_USER_ID": "1"
      },
      "disabled": false,
      "autoApprove": [
        "bitrix24_get_leads",
        "bitrix24_get_deals",
        "bitrix24_get_contacts",
        "bitrix24_get_companies"
      ]
    }
  }
}
```

### Переменные окружения

- `BITRIX24_WEBHOOK_URL` - URL вашего webhook в Bitrix24
- `BITRIX24_USER_ID` - ID пользователя (обычно 1 для администратора)

```bash

0 9 * * * /usr/bin/python3 /path/to/automation/mcp_scenarios.py daily_lead_analysis
```

## Безопасность

### Ограничение доступа

В `mcp.json` используйте `autoApprove` только для безопасных операций чтения:

```json
{
  "autoApprove": [
    "bitrix24_get_leads",
    "bitrix24_get_deals",
    "bitrix24_get_contacts"
  ]
}
```

Операции записи (create, update, delete) требуют подтверждения.

### Ротация webhook

Регулярно обновляйте webhook URL в Bitrix24 для безопасности.

## Troubleshooting

### MCP сервер не запускается

```bash
# Проверьте установку uvx
uvx --version

# Переустановите сервер
uvx --reinstall bitrix24-mcp-server
```

### Ошибка подключения к Bitrix24

Проверьте:
1. Правильность BITRIX24_WEBHOOK_URL
2. Права webhook (должны быть права на CRM)
3. Доступность Bitrix24 из вашей сети

### AI не видит MCP инструменты

1. Перезапустите Kiro
2. Проверьте `.kiro/settings/mcp.json`
3. Убедитесь, что `"disabled": false`
