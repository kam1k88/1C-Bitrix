# Интеграция с 1С: Подробное руководство

## Архитектура интеграции

```
AI Platform <--HTTP--> 1С HTTP-сервис <--> База 1С
```

## Настройка HTTP-сервиса в 1С

### Шаг 1: Создание HTTP-сервиса

1. Откройте конфигуратор 1С
2. Добавьте новый HTTP-сервис: "Общие" → "HTTP-сервисы"
3. Имя: `integration`
4. Корневой URL: `integration`

### Шаг 2: Создание методов

### Шаг 3: Настройка аутентификации

1. В HTTP-сервисе включите базовую аутентификацию
2. Создайте пользователя для интеграции
3. Установите права доступа

### Шаг 4: Публикация сервиса

1. Опубликуйте базу на веб-сервере
2. URL будет: `http://server:port/base_name/hs/integration/`
3. Проверьте доступность через браузер

## Примеры использования из Python

### Получение контрагентов

```python
from onec.client import OneCClient

client = OneCClient()
counterparties = client.get_counterparties({"phone": "+79001234567"})
print(counterparties)
```

### Создание заказа

```python
order_data = {
    "client_id": "uuid-клиента",
    "products": [
        {"product_id": "uuid-товара", "quantity": 10, "price": 100},
        {"product_id": "uuid-товара-2", "quantity": 5, "price": 200}
    ]
}

result = client.create_order(order_data)
print(f"Заказ создан: {result['order_id']}")
```

## Troubleshooting

### Ошибка 401 Unauthorized
- Проверьте логин/пароль
- Убедитесь, что пользователь имеет права

### Ошибка 500 Internal Server Error
- Проверьте логи 1С
- Убедитесь в правильности структуры данных

### Медленные запросы
- Добавьте индексы в базу 1С
- Оптимизируйте запросы
- Используйте кэширование
