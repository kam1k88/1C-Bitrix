# Руководство по Telegram ботам

## Обзор

Система включает три специализированных Telegram бота для автоматизации работы с Bitrix24 и 1С:

### 1. Analytics Bot 📊
**Для аналитиков и руководителей**

Функции:
- 📊 Отчет по остаткам товаров в 1С
- 💰 Отчет по продажам за период
- 👥 Топ клиентов по выручке
- 📈 Анализ конверсии лидов
- 📋 Кастомные отчеты
- 🤖 AI-анализ и рекомендации

Команды:
```
/start - Главное меню
/остатки - Отчет по остаткам
/продажи [период] - Отчет по продажам
/топ_клиенты [N] - Топ N клиентов
/конверсия - Анализ конверсии
/отчет - Создать кастомный отчет
/help - Справка
```

Примеры использования:
```
/продажи сегодня
/продажи неделя
/топ_клиенты 10
```

### 2. Sales Bot 💼
**Для менеджеров по продажам**

Функции:
- 📋 Просмотр активных лидов
- ➕ Создание новых лидов
- 📄 Генерация коммерческих предложений
- ✅ Управление задачами
- 🔍 Поиск клиентов в базе
- ⭐ AI-оценка качества лидов

Команды:
```
/start - Главное меню
/лиды - Мои активные лиды
/создать_лид - Создать новый лид
/кп - Сгенерировать КП
/задачи - Мои задачи
/клиент [имя] - Найти клиента
/help - Справка
```

Примеры использования:
```
/клиент ООО Продукты
/создать_лид
```

### 3. Support Bot 🛠️
**Для технической поддержки**

Функции:
- ❓ База знаний (FAQ)
- 🐛 Регистрация проблем
- 📊 Статус работы систем
- 📖 Инструкции по настройке
- 🤖 AI-помощник для решения проблем

Команды:
```
/start - Главное меню
/faq - Часто задаваемые вопросы
/проблема [описание] - Сообщить о проблеме
/статус - Статус систем
/инструкция [тема] - Получить инструкцию
/help - Справка
```

Примеры использования:
```
/проблема Не работает синхронизация с 1С
/инструкция настройка webhook
```

## Установка и настройка

### Шаг 1: Создание ботов в Telegram

1. Откройте @BotFather в Telegram
2. Создайте три бота:

```
/newbot
Имя: Analytics Bot
Username: your_analytics_bot

/newbot
Имя: Sales Bot
Username: your_sales_bot

/newbot
Имя: Support Bot
Username: your_support_bot
```

3. Сохраните токены каждого бота

### Шаг 2: Установка зависимостей

```bash
pip install python-telegram-bot==20.7
```

Или добавьте в `requirements.txt`:
```
python-telegram-bot==20.7
```

### Шаг 3: Настройка переменных окружения

Добавьте в `.env`:

```env
# Telegram Bot Tokens
ANALYTICS_BOT_TOKEN=your_analytics_bot_token_here
SALES_BOT_TOKEN=your_sales_bot_token_here
SUPPORT_BOT_TOKEN=your_support_bot_token_here

# Bitrix24 (уже есть)
BITRIX24_WEBHOOK_URL=https://your-domain.bitrix24.ru/rest/1/xxxxx/

# OpenAI (уже есть)
OPENAI_API_KEY=sk-your-key-here

# Anthropic (уже есть)
ANTHROPIC_API_KEY=sk-ant-your-key-here

# 1С (уже есть)
ONEC_BASE_URL=http://your-1c-server:port
ONEC_USERNAME=your_username
ONEC_PASSWORD=your_password
```

### Шаг 4: Запуск ботов

**Вариант 1: Запуск всех ботов сразу**
```bash
python bots/bot_manager.py
```

**Вариант 2: Запуск отдельных ботов**
```bash
# Analytics Bot
python bots/analytics_bot.py

# Sales Bot
python bots/sales_bot.py

# Support Bot
python bots/support_bot.py
```

**Вариант 3: Запуск в фоне (Linux/macOS)**
```bash
nohup python bots/bot_manager.py > bots.log 2>&1 &
```

**Вариант 4: Systemd сервис (Linux)**

Создайте `/etc/systemd/system/telegram-bots.service`:

```ini
[Unit]
Description=Telegram Bots for Bitrix24 and 1C
After=network.target

[Service]
Type=simple
User=your_user
WorkingDirectory=/path/to/project
Environment="PATH=/path/to/venv/bin"
ExecStart=/path/to/venv/bin/python bots/bot_manager.py
Restart=always

[Install]
WantedBy=multi-user.target
```

Запуск:
```bash
sudo systemctl enable telegram-bots
sudo systemctl start telegram-bots
sudo systemctl status telegram-bots
```

## Troubleshooting

### Бот не отвечает

1. Проверьте токен в `.env`
2. Проверьте, запущен ли бот: `ps aux | grep bot`
3. Проверьте логи: `tail -f bots.log`

### Ошибка подключения к Bitrix24

1. Проверьте `BITRIX24_WEBHOOK_URL` в `.env`
2. Проверьте права webhook в Bitrix24
3. Проверьте доступность Bitrix24: `curl $BITRIX24_WEBHOOK_URL/crm.lead.list`

### Ошибка подключения к 1С

1. Проверьте `ONEC_BASE_URL` в `.env`
2. Проверьте доступность 1С: `curl $ONEC_BASE_URL/hs/counterparties`
3. Проверьте логи 1С

### AI не работает

1. Проверьте `OPENAI_API_KEY` и `ANTHROPIC_API_KEY`
2. Проверьте баланс на аккаунтах OpenAI/Anthropic
3. Проверьте лимиты API

## Безопасность

### Рекомендации

1. **Не публикуйте токены** - храните в `.env`, добавьте в `.gitignore`
2. **Ограничьте доступ** - используйте whitelist пользователей
3. **Логируйте действия** - все операции записываются в логи
4. **Регулярно обновляйте** токены и пароли
5. **Используйте HTTPS** для всех API

### Whitelist пользователей

Добавьте в каждый бот:

```python
ALLOWED_USERS = [123456789, 987654321]  # Telegram user IDs

async def check_access(update: Update):
    user_id = update.effective_user.id
    if user_id not in ALLOWED_USERS:
        await update.message.reply_text("❌ Доступ запрещен")
        return False
    return True
```
