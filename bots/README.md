# Telegram Боты для Bitrix24 и 1С

## Три бота для автоматизации

### 📊 Analytics Bot
**Для аналитиков**
- Отчеты по остаткам, продажам, конверсии
- Топ клиентов
- AI-анализ и рекомендации

### 💼 Sales Bot
**Для менеджеров**
- Работа с лидами
- Генерация КП
- Управление задачами
- Поиск клиентов

### 🛠️ Support Bot
**Для поддержки**
- FAQ база знаний
- Регистрация проблем
- Статус систем
- Инструкции

## Быстрый старт

### 1. Создайте ботов в Telegram

Откройте @BotFather и создайте 3 бота:

```
/newbot
Имя: Analytics Bot
Username: your_analytics_bot

/newbot
Имя: Sales Bot  
Username: your_sales_bot

/newbot
Имя: Support Bot
Username: your_support_bot
```

Сохраните токены!

### 2. Установите зависимости

```bash
pip install python-telegram-bot==20.7
```

### 3. Настройте .env

Добавьте токены в `.env`:

```env
ANALYTICS_BOT_TOKEN=123456:ABC-DEF...
SALES_BOT_TOKEN=123456:ABC-DEF...
SUPPORT_BOT_TOKEN=123456:ABC-DEF...
```

### 4. Запустите боты

**Все сразу:**
```bash
python bots/bot_manager.py
```

**По отдельности:**
```bash
python bots/analytics_bot.py
python bots/sales_bot.py
python bots/support_bot.py
```

## Использование

### Analytics Bot

```
/start - Главное меню
/остатки - Отчет по остаткам
/продажи неделя - Продажи за неделю
/топ_клиенты 10 - Топ 10 клиентов
/конверсия - Анализ конверсии

Или просто напишите:
"Сколько продали за вчера?"
"Какие товары заканчиваются?"
```

### Sales Bot

```
/start - Главное меню
/лиды - Мои лиды
/создать_лид - Создать лид
/кп - Сгенерировать КП
/задачи - Мои задачи
/клиент ООО Продукты - Найти клиента

Или просто напишите:
"Создай лид для Ивана Петрова"
"Сгенерируй КП на молочку"
```

### Support Bot

```
/start - Главное меню
/faq - Часто задаваемые вопросы
/проблема Не работает синхронизация
/статус - Статус систем
/инструкция настройка webhook

Или просто напишите:
"Как создать лид в Bitrix24?"
"Не работает синхронизация с 1С"
```

## Troubleshooting

**Бот не отвечает:**
1. Проверьте токен в `.env`
2. Убедитесь, что бот запущен
3. Проверьте логи

**Ошибки API:**
1. Проверьте ключи OpenAI/Anthropic
2. Проверьте подключение к Bitrix24
3. Проверьте подключение к 1С
