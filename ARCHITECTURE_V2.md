## Компоненты системы

### 1. Bitrix24 Чат-боты (Open Lines)

**Analytics Assistant** 📊
- Отчеты по продажам
- Анализ остатков
- Конверсия лидов
- Топ клиенты

**Sales Assistant** 💼
- Помощь менеджерам
- Генерация КП
- Работа с лидами
- Поиск информации

**Warehouse Assistant** 📦
- Управление складом
- Контроль остатков
- Заказы поставщикам
- Инвентаризация

**BI Assistant** 📈
- Бизнес-аналитика
- Прогнозирование
- Дашборды
- KPI мониторинг

### 2. Коннекторы маркетплейсов

**Ozon Connector**
- Синхронизация товаров
- Обработка заказов
- Управление остатками
- Аналитика продаж

**Wildberries Connector**
- Загрузка товаров
- Обработка заказов
- Отслеживание поставок
- Финансовые отчеты

**Яндекс.Маркет Connector**
- Управление каталогом
- Обработка заказов
- Ценообразование
- Отзывы и рейтинги

### 3. AI-Ассистенты

**Для аналитиков:**
- Автоматические отчеты
- Прогнозирование продаж
- Анализ трендов
- Рекомендации

**Для менеджеров:**
- Оценка лидов
- Генерация КП
- Подсказки по сделкам
- Автоматизация рутины

**Для склада:**
- Оптимизация остатков
- Прогноз закупок
- Контроль сроков
- Автозаказы

### 4. Автоматизация (Bitrix24 Automation)

**Триггеры:**
- Новый заказ с маркетплейса → Создание сделки
- Низкий остаток → Уведомление + Автозаказ
- Новый лид → AI-анализ + Назначение менеджера
- Сделка закрыта → Синхронизация с 1С

**Роботы:**
- AI-оценка лидов
- Генерация документов
- Отправка уведомлений
- Обновление данных
